# 🏠 Propfolio

> Our personal property portfolio tracker — built for Australians, by us.

Propfolio is a private web app for tracking, analysing, and growing our property investment portfolio across Australia. From first purchase to full portfolio, everything in one place.

---

## 🎯 What We're Building

A simple, powerful tool to help us make smarter property decisions — tracking what we own, what we're watching, and where we want to go.

---

## ✅ Planned Features

### Phase 1 — The Basics
- [ ] Property listing (address, purchase price, purchase date)
- [ ] Current estimated value tracking
- [ ] Equity calculator (value minus mortgage remaining)
- [ ] Rental income tracker
- [ ] Simple portfolio summary dashboard

### Phase 2 — Analysis Tools
- [ ] Yield calculator (rental return %)
- [ ] Capital growth tracking over time
- [ ] Mortgage repayment breakdown
- [ ] Expense tracker (rates, insurance, maintenance)
- [ ] Cash flow summary (income vs expenses per property)

### Phase 3 — Growth Planning
- [ ] Target portfolio goals (e.g. 5 properties by 2030)
- [ ] Borrowing capacity estimator
- [ ] Suburb research notes
- [ ] Document storage (contracts, reports, insurance)
- [ ] AI-powered insights and suggestions

## 🛠️ Tech Stack

- **Frontend:** Next.js + Tailwind CSS
- **Backend:** Supabase (database + auth)
- **Hosting:** Vercel
- **Charts:** Recharts

---

## 🇦🇺 Australian Context

Built with Australian property investing in mind:
- Prices in AUD
- State-based stamp duty awareness
- Negative gearing considerations
- Australian lending landscape (LVR, LMI, offset accounts)

---

## 🔀 Git Workflow

- All development is done on `claude/` feature branches (e.g. `claude/new-session-1Sz2q`)
- Claude never merges to `main` or pushes to `main` directly
- After testing on the Claude branch, Stuart manually merges into `main`
- Vercel production deploys from `main` only

## 🚀 Getting Started

```bash
git clone https://github.com/comfybear71/propfolio
cd propfolio
npm install
npm run dev
```
